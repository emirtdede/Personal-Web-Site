# English Version

A modern, highly interactive, and visually polished personal portfolio website designed to showcase software development projects, technical skills, and academic accomplishments. Tailored with a custom premium dark-rose theme and optimized for fast client-side performance.

🌐 **Live Demo**: [emirtdede.com](https://emirtdede.com)

---

## 🚀 Key Features

*   **🌐 Localization & Translation Engine**: Integrates a dynamic language switcher toggling between Turkish (TR) and English (EN) that preserves user selections locally (`localStorage`).
*   **🎨 Premium Dark-Rose Aesthetic**: Custom styled with deep rose accents (`#cc9595`) over a dark structural background (`#342626`). Features modern web typography, fluid grids, and sleek hover transition details.
*   **📂 Interactive Project Showroom**: Displays software projects with dedicated tags indicating the tech stack. Includes direct triggers to access repositories, live demonstrations, or design files.
*   **✉️ Serverless AJAX Contact Form**: Submits messages directly using Formspree API. Includes a custom drag-and-drop attachment uploader (up to 10MB limit) with built-in validation.
*   **🔔 Glassmorphic Notification Modals**: Replaces basic alert dialogues with responsive overlay cards matching the dark-rose design theme.

---

## 🛠️ Technology Stack

*   **Frontend**: HTML5, Vanilla CSS3 (custom animations and theme attributes), Vanilla JavaScript (DOM manipulation and storage wrappers)
*   **Integrations**: Formspree (serverless form handling)

## ⚙️ Installation & Running

Since the project is built entirely using static client-side technologies, it requires no build compilations.

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/emirtdede/Personal-Web-Site.git
    cd Personal-Web-Site
    ```
2.  **Launch Locally**:
    Double-click `index.html` to run in any browser, or spin up a quick Python local server:
    ```bash
    python -m http.server 8000
    ```
    Open `http://localhost:8000` in your browser.

# Türkçe Versiyon

Yazılım projelerini, teknik yetkinlikleri ve akademik çalışmaları sergilemek üzere tasarlanmış, modern ve etkileşimli bir kişisel portföy web sitesidir. Özel koyu-gül (dark-rose) tasarım teması ile şekillendirilmiş olup, tarayıcı tarafında yüksek performansla çalışacak şekilde optimize edilmiştir.

🌐 **Canlı Demo**: [emirtdede.com](https://emirtdede.com)

---

## 🚀 Öne Çıkan Özellikler

*   **🌐 Dinamik Dil Çeviri Sistemi**: Türkçe (TR) ve İngilizce (EN) dilleri arasında anlık geçiş sağlayan ve kullanıcı tercihini tarayıcı hafızasında (`localStorage`) saklayan sistem.
*   **🎨 Premium Koyu-Gül Estetiği**: Koyu kahverengi arka plan (`#342626`) üzerinde yumuşak gül detayları (`#cc9595`) ile hazırlanan özgün tasarım. Akıcı geçişler, Poppins yazı tipi ve estetik kaydırma çubukları içerir.
*   **📂 Etkileşimli Proje Vitrini**: Geliştirilen projeleri kullanılan teknolojilere göre etiketleyip listeleyen, GitHub kaynak kodlarına veya Figma tasarımlarına doğrudan yönlendirme içeren kart yapısı.
*   **✉️ Sunucusuz AJAX İletişim Formu**: Formspree API aracılığıyla doğrudan mesaj gönderimi. Maksimum 10MB boyutunda dosya yükleme desteği ve istemci tarafında dosya doğrulama sistemi.
*   **🔔 Arayüzle Uyumlu Uyarı Panelleri**: Tarayıcının varsayılan uyarı pencereleri yerine, koyu-gül temasına uygun olarak tasarlanmış glassmorphic modal pencereleri.
